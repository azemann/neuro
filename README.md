# NeuroBrand v1.5

Application fullstack pour générer des mini-marques à partir de tendances culturelles.  
🧠 Client : React + Vite  
🔧 Backend : Express.js  
☁️ Déploiement : local, Replit, Render, Vercel

## Installation locale

```bash
git clone <repo>
cd neurobrand
bash start.sh
```

Accès frontend : http://localhost:3000  
API backend : http://localhost:5000/generate
